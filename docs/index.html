재고표에서 금일생산, 생산목표 수치를 클릭하면 직접 수정할 수 있습니다. Enter로 저장, ESC로 취소합니다.

하루 작업이 끝나면 금일마감 버튼을 눌러 당일 생산/출하 실적을 저장합니다. 최근 7일간의 마감 기록을 확인할 수 있습니다.
